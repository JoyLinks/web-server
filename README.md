# JOYZL Server

JOYZL Server 是纯粹中国智造，完全开源的国产化服务器软件；

支持基于HTTP协议的WEB服务；
支持基于HTTP协议的WEBDAV服务；

正在开发中。。。

### 语言代码（Language Codes）

由 [www.iso639-3.sil.org](http://www.iso639-3.sil.org/) 提供 ISO 639-3 语言代码集
